## 專案內容

- vue
- scss
- 使用CSS變數
- grid排版
- 新增額外自訂電量功能

## 功能介紹

- 裝置充電中會顯示充電中提示
- 裝置電量大於99%會顯示滿電提示
- 裝置電量小於20%會顯示電量低提示
- 電池顏色會隨電量顯示各階段顏色
